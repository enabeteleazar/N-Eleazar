## À propos de moi

👋 Bonjour, je suis NABET Eleazar, un développeur web front-end de 36 ans. Passionné par l’innovation et l’amélioration continue, je m’efforce de créer des expériences web exceptionnelles en utilisant les dernières technologies front-end.

Toujours en quête de nouvelles connaissances, je suis également autodidacte et j’aime me former sur les dernières technologies et tendances du web. Mon intérêt pour l’informatique et le code remonte à ma jeunesse, lorsque j’ai découvert ce monde fascinant.
